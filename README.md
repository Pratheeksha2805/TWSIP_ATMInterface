# TWSIP_ATMInterface
ATM Interface
Welcome to the ATM System! This Java program simulates basic ATM functionalities such as checking balance, withdrawing, depositing, and transferring funds between accounts.

Project Overview
The ATM System project consists of a command-line application written in Java. It allows users to perform various banking operations after successful authentication.

Features
User Authentication: Users are required to enter their account number and PIN for authentication.

Main Menu:

Check Balance: View account balance.
Withdraw: Withdraw funds from the account.
Deposit: Deposit funds into the account.
Transfer: Transfer funds to another account.
Exit: Exit the ATM System.

Follow On-screen Instructions:

Enter your account number and PIN for authentication.
Choose from the main menu options to perform banking operations.
User Authentication
The system comes pre-loaded with three accounts for demonstration purposes.
Account Numbers: "123456", "789012", "345678"
Corresponding PINs: "1111", "2222", "3333"
Initial Balances: $1000.0, $2000.0, $1500.0
Customize the Project
You can customize the project by modifying account details, PINs, and initial balances in the ATMSystem class.
Dependencies
No external dependencies are required.
